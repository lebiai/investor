## 它能帮你做什么

### 📋 内容梳理
扔过来一篇文章、PDF、BP、会议录音，说 *"帮我整理这个"*，它会：
- 转成可读的文本
- 提炼核心信息
- 自动归档，下次直接调

### 🔍 赛道分析
想深入了解一个行业，说 *"分析一下固态电池赛道"*，它会：
- 全景扫描（产业链、政策、国际对标）
- 多维度打分（市场规模、增速、竞争格局……）
- 拉出赛道里的公司清单
- 输出分析报告，存入知识库

### 📄 BP 分析
拿到一份 BP，说 *"帮我看这个BP"*，它会：
- 六个角色轮流审阅（行业研究员、财务、风控、产业资本、竞争对手、投委会主席）
- 交叉辩论，找共识和分歧
- 输出：速览卡 + 评分雷达 + 信号表 + 估值区间 + 尽调路线图

### 🎯 项目扫描
想扫一遍某个领域有什么项目，说 *"帮我扫描一下半导体材料领域"*，它会：
- 多源搜索（36氪、投资界、IT桔子……）
- 结构化提取项目信息
- 自动加入盯盘清单

### 💰 融资事件
想知道最近谁融了资，说 *"搜一下上周AI融资"* 或 *"本月新能源融资汇总"*，它会：
- 搜索全网融资信息
- 按时间/赛道/轮次整理
- 自动存储，下次查历史直接出

### 📊 盯盘跟踪
关注某个赛道或公司，说 *"关注固态电池"*，之后问 *"固态电池有什么新动态"*，它会：
- 增量更新，只告诉你变了什么
- 自动对比上次结果，输出变化摘要

### 🏢 投后管理
管理你的组合公司，说 *"我的投资组合"*，它会：
- 里程碑追踪 + 预警
- 董事会准备材料
- 退出监控

### ✍️ 创作内容
基于你的知识库写东西，说 *"帮我写一篇关于固态电池的文章"* 或 *"写一份IC备忘录"*

### 📎 文件模板
说 *"把这个保存为模板"* → 以后说 *"用会议纪要模板生成一份"* → 自动填好输出 Word/PPT/Excel

### 📝 会议纪要
说 *"帮我记一下今天的会"*，它会整理成结构化纪要，联系人自动入人脉库

### 👥 人脉管理
说 *"帮我记一下张总，红杉资本的"*，下次说 *"查一下张总"* 直接调出档案

### 🌱 个人成长
说 *"记录一下"* 或 *"周报"*，它会自动捕获你的洞察，每周蒸馏成成长记录
