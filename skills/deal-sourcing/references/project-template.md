# 项目档案模板

写入 `data/companies/` 时的公司档案格式。
文件头部必须加 YAML header（参考 `docs/DATA-STANDARD.md` §3.2）：

```
# [公司名]

## 基础信息
- **成立时间**：
- **主营业务**：
- **最新估值/市值**：
- **融资轮次**：
- **主要投资方**：

## 核心数据
| 指标 | 最新 | 前年 | 说明 |
|------|------|------|------|

## 知识来源
- YYYY-MM-DD：[来源 + 操作]
- ...

## 关键洞察
- （所有相关 skill 的累积观点）
```
