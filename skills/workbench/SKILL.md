# 投资人工作台

## 触发词

- "开工" / "投资助手" / "有什么能力" — 展示功能菜单
- "初始化" / "安装" / "配置" — 首次部署时一键初始化

## 初始化工作流

用户说"初始化"：

1. 告知用户："正在初始化投资人技能套件，请稍候..."
2. 执行初始化流程（详见 [skill:init]）
3. 所有检查→下载→注册步骤自动完成
4. 输出初始化结果报告
5. 告知用户初始化完成，建议新建会话使用
6. 输出功能菜单（见下方）

## 路由规则

- 关键词含"初始化""安装""配置" → 执行初始化流程
- 关键词含"分析""赛道""行业""怎么看" → 激活 [skill:sector-analysis]
- 关键词含"整理""总结""摘要""读一下" → 激活 [skill:research-digest]
- 关键词含"模板""套模板""生成文档""保存模板""模板化" → 激活 [skill:template-prod]
- 关键词含"写""发""文章""小红书""朋友圈""备忘录""IC memo" → 激活 [skill:content-prod]
- 关键词含"扫描""项目""标的""融资" → 激活 [skill:deal-sourcing]
- 关键词含"关注""新动态""盯盘" → 激活 [skill:watch]
- 关键词含"BP""审阅""路演""Term Sheet""评估项目""商业计划书""评估" → 激活 [skill:deal-review]
- 关键词含"投资组合""组合公司""投后""看板""里程碑""董事会""退出""预警" → 激活 [skill:portfolio-tracker]
- 关键词含"周报""记录一下""成长""总结""复盘" → 执行 personal-growth 蒸馏流程
- 关键词含"记一下""会议""纪要""录音""路演纪要""投委会纪要" → 激活 [skill:meeting-notes]
- 关键词含"联系人""人脉""记一下[姓]""查一下[姓]""认识谁""见过的人" → 激活 [skill:contact-crm]
- 无法匹配 → 根据上下文做最佳判断或询问用户

### 复合意图处理

当用户一句话同时触发多个 skill 时（如"分析半导体设备并写成文章"）：

1. 按顺序执行：优先执行数据收集类 skill（sector-analysis / deal-sourcing / research-digest / deal-review），再执行产出类 skill（content-prod / template-prod）
2. 告知用户："将先进行[分析]，再基于结果[产出]"
3. 前一个 skill 完成后，将输出结果作为下一个 skill 的输入上下文
4. 最终交付完整结果
