# 联系人搜索规则

> 定义按行业/角色/公司等维度搜索联系人的标准化规则。

## 搜索维度优先级

用户查询 → 分解为以下维度 → 逐层缩小范围：

| 维度 | 示例值 | 搜索方式 |
|------|--------|---------|
| 行业/赛道 | 固态电池、医疗、AI | rg 搜索 tags/行业字段 |
| 角色 | founder、investor、expert | rg 搜索 role 字段 |
| 公司 | XX科技 | rg 搜索 company 字段 |
| 姓名 | 张三 | 直接文件名匹配 |
| 介绍人 | 李四 | rg 搜索 关系网络字段 |

## 模糊匹配规则

当用户输入不精确时（如"做半导体的FA"）：

1. 拆分为：行业=半导体，角色=FA
2. 对每个维度执行 rg 搜索
3. 取交集

## 最近互动优先

搜索结果默认按最近互动日期倒序排列，确保活跃联系人优先展示。

## 空白结果降级

如搜索结果为空，自动降级：
1. 去掉行业限制 → 仅按角色搜 → "该行业暂无[角色]，但有其他行业的[角色]：..."
2. 去掉角色限制 → 仅按行业搜 → "该行业暂无[角色]，但有该行业的其他联系人：..."
3. 全空 → "联系人库中暂无匹配，说'帮我记一下...'新增"
