# 董事会准备模板

> 用于生成董事会简报。整合公司档案 + 赛道数据 + 外部信息。

## 简报结构

### 一、公司整体状态卡

| 项目 | 数据 | 趋势 |
|------|------|------|
| 当前估值 | ¥XXXX万 | ↑/↓/→ |
| 投资倍数 | X.Xx MOIC | ↑/↓/→ |
| 里程碑进度 | X/Y 完成 | 🟢🟡🔴 |
| 最新营收（季度） | ¥XXXX万 | ↑/↓/→ |
| 毛利 | X% | ↑/↓/→ |
| 现金流 | X个月 | ↑/↓/→ |
| 预警状态 | 🟢🟡🔴 | — |

### 二、经营回顾

按季度展开：
- 营收拆解：老客户复购 / 新客户贡献
- 费用结构：研发 / 销售 / 管理
- 关键指标：CAC / LTV / 留存率 / NPS
- 团队变化：人数 / 核心岗位

### 三、核心里程碑回顾

| 里程碑 | 预期 | 实际 | 偏差 | 原因 |
|--------|------|------|------|------|
| ... | ... | ... | ... | ... |

### 四、需董事会决策事项

| 事项 | 背景 | 建议 | 决策类型 |
|------|------|------|---------|
| ... | ... | ... | 批准/讨论/知悉 |

### 五、外部环境

- 赛道趋势变化
- 竞品动态
- 监管/政策更新
- 潜在机会/威胁

### 六、投资人关注点

1. 最大风险点
2. 最大机会点
3. 下季度关键目标
4. 需要创始人回答的问题清单

## 信息聚合顺序

1. 先读公司档案（最近状态）
2. 再读赛道档案（行业背景）
3. [skill:agent-reach]搜索竞品动态
4. 整合生成
