# 预警规则库

> 定义投后预警的触发条件和严重度评级。基于组合公司状态 + 外部信息自动检测。

## 预警等级

| 等级 | 定义 | 响应时间 | 标记 |
|------|------|---------|------|
| 🔴 高危 | 威胁公司生存或重大投资损失 | 24小时内 | 推送到看板顶部 |
| 🟠 重要 | 显著偏离预期，需管理层关注 | 1周内 | 在看板标记 |
| 🟡 关注 | 潜在风险信号，需持续观察 | 1个月内 | 记录到预警列表 |

## 🔴 高危预警规则

| 规则 | 触发条件 | 检测方式 | 示例 |
|------|---------|---------|------|
| 现金流枯竭 | 现金流 < 3个月且无融资进展 | 档案财务数据 + [agent-reach]搜索融资新闻 | 公司A现金仅够2个月 |
| 核心创始人离职 | CEO/CTO/创始人离职 | [agent-reach]搜索 + 用户输入 | CTO宣布离职 |
| 关键对赌未达成 | 投资条件中的对赌条款未达成 | 里程碑检测 | 营收对赌未完成 |
| 重大法律风险 | 起诉/监管调查/专利纠纷 | [agent-reach]搜索 | 被竞争对手起诉 |
| 欺诈信号 | 财务数据异常/创始人不回应 | 用户输入 + 交叉验证 | 连续3个月不提供财务数据 |

## 🟠 重要预警规则

| 规则 | 触发条件 | 检测方式 |
|------|---------|---------|
| 连续里程碑延迟 | ≥2个里程碑延迟 >1个月 | 里程碑引擎自动检测 |
| 下一轮融资困难 | 计划融资时间已过2个月未关闭 | [agent-reach]搜索 + 档案比对 |
| 核心客户流失 | 大客户流失/合同不续签 | 用户输入 |
| 高管离职 | C-level/VP级离职（非创始人） | [agent-reach]搜索 + 用户输入 |
| 营收连续不达标 | 连续2个季度营收低于预期 >20% | 财务追踪自动比对 |
| 竞品重大进展 | 竞品获得大额融资/发布重要产品 | [agent-reach]搜索 |
| 股东纠纷 | 股东间出现重大分歧 | 用户输入 |

## 🟡 关注预警规则

| 规则 | 触发条件 | 检测方式 |
|------|---------|---------|
| 单次里程碑延迟 | 1个里程碑延迟 <1个月 | 里程碑引擎检测 |
| 普通岗位离职 | 非核心岗位人员变动增加 | 用户输入 |
| 行业政策变化 | 行业监管/政策有重大变化 | [agent-reach]搜索 |
| 估值倒挂 | 下一轮估值低于本轮 | [agent-reach]搜索同赛道估值 |
| 烧钱率上升 | 月度开支显著增加 | 财务追踪 |
| 规模化瓶颈 | 增速放缓但未见改善措施 | 里程碑 + [agent-reach]搜索 |

## 预警生命周期

```
发现 → 评估定级 → 记录到 alerts.md → 在看板展示
  ↓
关闭（问题解决）→ 标记为已关闭，移入 history
升级（问题恶化）→ 升一级
降级（风险缓解）→ 降一级
```

## 预警去重

同一公司同一预警规则：
- 7天内不重复触发
- 跨周期的相同预警 → 更新"最近确认"时间，不新增行
