# 退出监控

> 跟踪组合公司的退出路径进展。支持 IPO / 并购 / 二级出售三种路径。

## 退出路径类型

### 一、IPO

| 阶段 | 典型时长 | 关键信号 | 追踪点 |
|------|---------|---------|--------|
| 前期准备 | 6-12个月 | 聘请CFO / 引入保荐人 / 审计启动 | 券商选择 / 审计进度 |
| 申报期 | 3-6个月 | 提交招股书 / 交易所受理 | 招股书披露 / 监管问询 |
| 发行期 | 1-3个月 | 路演 / 定价 / 挂牌 | 超额认购倍数 / 定价区间 |
| 锁定期 | 6-12个月 | 股价表现 / 减持窗口 | 股价 vs 发行价 / 交易量 |

### 二、并购

| 阶段 | 关键信号 | 追踪点 |
|------|---------|--------|
| 初步接触 | 买方接触 / 签署NDA | 买方背景 / 估值预期 |
| 尽调 | 买方入场尽调 | 尽调范围 / 预计时长 |
| 谈判 | 条款书谈判 / 价格博弈 | 交易结构 / 支付方式 |
| 交割 | 签署SPA / 完成交割 | 交割条件 / 监管审批 |

### 三、二级出售

| 阶段 | 关键信号 | 追踪点 |
|------|---------|--------|
| 评估 | 股东决定退出 | 估值预期 / 买家寻找 |
| 交易 | 与买家谈判 | 折扣率 / 交易结构 |
| 交割 | 完成过户 | 资金到账 |

## 退出健康度评分

| 维度 | 评分标准 |
|------|---------|
| 退出路径明确性 | 有明确路径和时间表 → 🟢 / 路径模糊 → 🟡 / 无规划 → 🔴 |
| 进展 vs 计划 | 按时推进 → 🟢 / 延迟 <3个月 → 🟡 / 延迟 >3个月 → 🔴 |
| 市场窗口 | 市场环境有利 → 🟢 / 不确定 → 🟡 / 市场关闭 → 🔴 |
| 公司表现 | 达到上市/并购标准 → 🟢 / 有差距但可弥合 → 🟡 / 差距大 → 🔴 |

## 退出回报预测

计算逻辑：
- 投资金额 × 预期估值倍数 × 稀释调整 × 退出概率
- 显示乐观/基准/悲观三个场景

```
预期回报：
├── 乐观：¥XXXX万（X.Xx）— 概率：30%
├── 基准：¥XXXX万（X.Xx）— 概率：50%
└── 悲观：¥XXXX万（X.Xx）— 概率：20%
```
