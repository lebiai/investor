# 文档分类器

## 分类规则

按文档内容和格式特征自动分类：

| 文档类型 | 关键特征 | 判断依据 |
|---------|---------|---------|
| BP / 商业计划书 | "商业计划书""business plan""融资计划""项目简介" 等文字型文档 | 文字为主，结构为：痛点→方案→市场→团队→财务 |
| Pitch Deck / 路演材料 | PPT 格式，"投资亮点""估值""增长策略""谢谢" | PPT 文件 + 路演特征短语 |
| Term Sheet / 投资条款 | "投资条款""估值""清算优先权""防稀释""董事会" | 法律/条款类文本结构 |
| 财务模型 / 财务预测 | 大量财务表格，"假设""预测""增长率""毛利率" | Excel 文件 或 以表格数据为主的文档 |
| 尽调报告 / DD | "尽调""due diligence""法务""财务""业务尽调" | 系统性评估语言 + 多维度分类 |
| 竞品分析 | 含竞争格局/竞品对比表，"对标""SWOT" | 对比分析类结构 |
| 其他 | 无法匹配以上特征 | — |

如文档已经过 research-digest 处理，可从其分类结果继承。
