# 个人成长系统

> 三层捕获：自动记录 → 收尾追问 → 每周蒸馏。越用越厚。

## 移植

```bash
cp -r personal-growth/ 新项目根目录/
# 打开新项目，直接可用
```

## 使用

日常无需额外操作，正常工作即可自动积累。
每周说一次"周报"完成定期梳理。

## 目录

- `vault/timeline.md` — 永久洞察记录（逐月增厚）
- `inbox/timeline.md` — 碎片临时区（每周清空）
- `context/timeline.md` — 自动捕获日志（无感知）
- `pulse/weekly/` — 周报归档
- `prompts/` — 追问模板
