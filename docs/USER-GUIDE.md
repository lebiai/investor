# 投资人助手 — 用户操作指南

> 这是一个 AI 投资助手。不是工具，是和你对话的搭档。
> 不需要学操作，只需要正常说话。

---

## 一、开始使用

### 第一次打开

说"开工"，看到功能菜单就绪。

### 之后每次

直接说需求就行。不用每次都说"开工"，系统会根据你说的话判断你要做什么。

---

## 二、8 个能力

### ① 内容梳理 — 知识库的入口

```
你说： "帮我整理一下这个文件"（上传 PDF/Word/PPT/Excel）
系统做： 读取内容 → 提取要点 → 归档 → 关联知识库
```

**之后你再做赛道分析时，系统会自动引用你上传过的资料。**

### ② 赛道深度研究

```
你说： "帮我分析一下固态电池"
系统做： 搜索信息 → 六维分析 → 六家机构评估 → 公司扫描 → 出报告
```

**分析完了系统会问** "有没有什么让你意外的发现？"
这一问不是为了客气——你说的收获会自动积累成你的个人洞察库。

**用好它：**
- 推荐先完整说"帮我分析一下[赛道名]"
- 分析完后可以追问"深入分析一下里面的XX公司"
- 或者"把这个赛道分析写成文章"

### ③ BP审阅 — 审阅项目材料

```
上传 BP/路演材料/Term Sheet 后说： "帮我看这个BP"
系统做： 六角色独立评估 → 辩论 → 出审阅报告 + 尽调路线图
```

审阅报告直接在会话中查看，不用翻文件。

### ④ 项目扫描 — 找标的

```
你说： "帮我扫描一下固态电池领域的新项目"
系统做： 查知识库 + 实时搜 → 出项目清单
```

### ⑤ 盯盘跟踪 — 持续关注

```
先说： "关注固态电池"
以后说： "固态电池有什么新动态"
系统自动比对上次结果，只输出变化的部分
```

### ⑥ 内容产出 — 写文章/报告

```
你说： "帮我写一篇关于固态电池的小红书"
系统做： 查知识库 → 套模板 → 出稿
```

### ⑦ 模板生成 — 生成 Office 文档

```
你说： "把这个文档模板化"（上传一份做好的文档）
系统做： 分析结构 → 提取模板 → 保存

之后说： "用[模板名]生成一份"
系统按模板生成 Word/PPT/Excel，自动打开
```

### ⑧ 个人成长 — 你的第二大脑

```
日常： 系统自动记录你做了什么分析、看了什么材料
复盘： 说"记录一下，我注意到..." 系统存入你的洞察库
周报： 每周末说"周报"，系统引导你花 5 分钟梳理本周收获
```

---

## 三、用好它的 3 个关键

### 1. 越用越厚

每次分析、每次审阅的结果都会自动存入知识库。下次你说"帮我分析一下XX"时，系统会先翻知识库里的已有资料，再上网搜新的。

一周后和第一天用的体验完全不同。

### 2. 追问是精华

每个能力完成后，系统会追问 1 句话。**这一问不是废话。** 你的回答会被存进个人洞察库，以后做类似分析时系统会说"你之前有一条相关的记录，要看吗？"

### 3. 每周 5 分钟

周末说"周报"，系统会把你这一周的碎片整理成永久记录。坚持 3 周，你会有一个越来越厚的个人洞察库。

---

## 四、常见问题

### 系统说"知识库里没找到" — 然后呢？

正常。知识库是慢慢积累的。第一次做赛道分析时数据是空的，全靠 agent-reach 实时搜索。第二次再做同一赛道时，系统会引用上次的结果。

### 我上传的文件存在哪了？

不用管。系统自动归档、自动标注来源。你需要的时候说"帮我找一下XX文件"就能找到。

### 系统没理解我的意思怎么办？

直接换一种说法。比如"分析一下固态电池"和"帮我看看固态电池这个方向"都会触发同一个能力。如果实在不匹配，说"开工"看菜单，按菜单上的话术说。

### 移植到其他项目？

整个 `personal-growth/` 目录可以复制到任何项目，里面自带 AGENTS.md，不需要额外配置。

---

## 五、一句话总结

> **正常说话，越用越厚。**
> 不需要学习操作，不需要记命令。
> 用 3 周，知识库和个人洞察会开始变得有价值。
