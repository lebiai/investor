# 输出规范

最终交付物包含四个部分：

## 1. 完整分析报告

结构：
```
# [赛道名] 赛道深度分析

## 执行摘要（300 字内）
一句话总结 + 核心发现 + 投资信号的最高优先级

## 一、赛道基本面
（六维分析摘要，每个维度 2-3 段）

## 二、多机构评估观点
（六家各自的核心观点，每家 3-5 句话）

## 三、公司层扫描
（上市公司对比表 + 未上市公司清单 + 产业链位置）

## 四、辩论分析
（共识/分歧/盲点）

## 五、综合投资建议
（信号类型 + 优先级排序 + 下一步）
```

## 2. 赛道打分卡

表格格式：
| 维度 | 评分(1-10) | 核心判断 | 分歧度 |
|------|-----------|---------|--------|
| 市场规模 | 7 | ... | 低 |
| 增速 | 8 | ... | 中 |
| ... | ... | ... | ... |

8 分以上代表真正卓越，大多数赛道各维度在 4-7 分。
