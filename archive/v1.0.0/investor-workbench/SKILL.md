# 投资人工作台

## 触发词

- "开工" / "投资助手" / "有什么能力" — 展示功能菜单
- "初始化" / "安装" / "配置" — 首次部署时一键初始化

## 初始化工作流

用户第一次打开项目时，如果技能未生效，说"初始化"：

1. 告知用户："正在初始化投资人技能套件，请稍候..."
2. 在终端执行初始化脚本：
   ```bash
   bash scripts/init.sh
   ```
3. 等待脚本完成
4. 告知用户初始化完成，建议新建会话

## 路由规则

根据用户意图路由到对应 skill 的工作流：

- 关键词含"初始化""安装""配置" → 执行初始化流程（bash scripts/init.sh）
- 关键词含"分析""赛道""行业""怎么看" → 激活 investor-sector-analysis
- 关键词含"整理""总结""摘要""读一下" → 激活 investor-research-digest
- 关键词含"写""发""文章""小红书""朋友圈" → 激活 investor-content-prod
- 关键词含"扫描""项目""标的""融资" → 激活 investor-deal-sourcing
- 关键词含"打开""浏览""知识库" → 输出知识库浏览指引
- 无法匹配 → 根据上下文做最佳判断或询问用户
