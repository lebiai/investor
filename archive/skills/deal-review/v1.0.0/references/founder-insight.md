# 创始人穿透分析

> 超级投资人读人。从文档中读出创始人的认知深度、执行力、诚实度和动机。

## 分析维度

### 维度1: 认知深度

| 信号 | 高认知表现 | 低认知表现 |
|------|-----------|-----------|
| 行业理解 | 有独到的第一性原理分析，非行业报告搬运 | 照搬第三方数据/行业通识，无个人见解 |
| 竞争认知 | 准确说出每个竞品的核心差异和自身定位 | "他们没有我们做得好" 但不展开 |
| 风险意识 | 主动提及3个以上真实风险及应对策略 | 只谈利好，不主动说风险 |
| 用户理解 | 有具体的用户画像/使用场景/痛点案例 | 用户描述笼统，无场景感 |

### 维度2: 执行力证据

| 信号 | 强执行力表现 | 弱执行力表现 |
|------|------------|------------|
| "已经" vs "计划" | "我们已经做到X（有数据）" | "我们计划做X" "下个季度准备做X" |
| 关键里程碑 | 有明确的阶段性成果（产品/客户/融资） | 只有时间轴没有里程碑 |
| 迭代速度 | "我们在X个月内从v1迭代到v3" | 没有迭代记录或很久未更新 |
| 团队建设 | 已有完整的关键团队/正在有条不紊搭建 | 核心岗位长期空缺或在所有关键角色 |

### 维度3: 诚实度信号

| 信号 | 高诚实度 | 低诚实度 |
|------|---------|---------|
| 主动暴露 | 主动说"我们还没解决X""竞品在Y比我们强" | 只说自己的好，不提短板 |
| 竞争描述 | "竞品X在A维度领先，我们选择在B维度差异化" | "我们是最好的""没有竞争对手" |
| 数据呈现 | 给出原始数据/计算口径/假设条件 | 只给结论不给推导过程 |
| 对质疑态度 | 文中预留可能被挑战的点并主动回应 | 回避争议话题，写得模糊/笼统 |

### 维度4: 动机判断

| 类型 | 特征 | 置信度 |
|------|------|--------|
| 使命驱动 | 对解决的问题有内在热情，谈痛点时情绪真实 | 通过cognitive mapping判断 |
| 财务驱动 | 强调退出节奏/估值倍数/类比成功项目 | 和赛道阶段是否匹配 |
| 虚荣驱动 | 过多强调媒体曝光/奖项/与名人合影 | 和业务核心指标不成比例 |
| 焦虑驱动 | 融资紧迫性强却说不出具体用途 | 和文档中资金使用计划矛盾 |

## 输出格式

```
## 创始人穿透分析

### 认知深度: [高/中/低]

- 置信度：[高/中/低]

### 执行力: [强/中/弱]
- 证据：[具体引用]
- 置信度：[高/中/低]

### 诚实度: [高/中/低]
- 证据：[具体引用]
- 置信度：[高/中/低]

### 动机判断: [使命/财务/虚荣/焦虑驱动]
- 证据：[具体引用]

### 综合评估
[一句话总结创始人的核心画像]
```
