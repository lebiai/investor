# 结构化提取

## 通用提取字段（所有文档类型均提取）

| 字段 | 说明 | 提取方式 |
|------|------|---------|
| 公司名 | 项目主体 | 直接提取 |
| 赛道/行业 | 所属领域 | 直接提取 |
| 一句话定位 | 公司做什么的 | 从开头/Slogan提取 |
| 融资阶段及金额 | 本轮融资金额+轮次 | 融资页提取 |
| 成立时间 | 公司成立时间 | 直接提取 |
| 创始人/团队 | 核心团队背景 | 团队页提取 |

## 按文档类型的特定提取

### BP 类额外提取

| 字段 | 提取来源 |
|------|---------|
| 市场空间（TAM/SAM/SOM） | 市场分析章节 |
| 目标用户/客户画像 | 用户/市场章节 |
| 核心产品/技术说明 | 产品章节 |
| 商业模式 | 盈利模式章节 |
| 竞品定位 | 竞争分析章节 |
| 历史融资及估值 | 融资历史 |
| 核心运营数据 | 如有提供 |

### Pitch Deck 类额外提取

| 字段 | 提取来源 |
|------|---------|
| 投资叙事主线 | Deck 逻辑链 |
| 视觉强调的数据 | 图表/大字 |
| 竞争壁垒主张 | 壁垒页 |
| 资金用途 | Use of Funds 页 |
| 关键里程碑 | Timeline 页 |

### Term Sheet 类额外提取

| 字段 | 提取来源 |
|------|---------|
| 估值 / 投前投后 | 条款页 |
| 清算优先权结构 | 清算条款 |
| 防稀释条款 | 防稀释条款 |
| 董事会构成 | 治理条款 |
| 对赌/里程碑条款 | 特殊条款 |
| 员工期权池 | 期权相关条款 |

### 财务模型类额外提取

| 字段 | 提取来源 |
|------|---------|
| 收入预测（逐年） | 预测表 |
| 毛利率/净利率假设 | 利润表假设 |
| 单人产出/获客成本等单位经济 | 运营指标表 |
| 现金流预测 | 现金流量表 |
| 估值假设 | 如果需要配合TS |

## 完整性评估

提取完成后标注：

```
[提取完整性]
✅ 团队信息齐全
✅ 市场空间有引用来源
⚠️ 财务数据仅覆盖1年（建议要求3年）
❌ 未提及竞争格局（重大缺失）
```
