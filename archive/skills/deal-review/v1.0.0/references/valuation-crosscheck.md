# 估值交叉验证

## 三种方法

### 方法1: 可比公司法

使用 [skill:agent-reach] 搜索可比上市公司/近期融资标的的估值数据：

```
可比公司估值区间：
| 公司 | 阶段 | 估值 | PS/PE | 与目标相似度 |
|------|------|------|-------|------------|
| [公司A] | [轮次] | [估值] | [PS/PE] | [高/中/低] |
| [公司B] | [轮次] | [估值] | [PS/PE] | [高/中/低] |
```

### 方法2: 近期交易法

搜索同赛道近期融资事件，按阶段对齐：

```
同赛道近期交易：
| 项目 | 轮次 | 融资金额 | 投后估值 | 时间 | 数据来源 |
|------|------|---------|---------|------|---------|
```

### 方法3: DCF 敏感度分析（如有财务预测）

```
DCF 敏感度分析：
假设                     → 估值的低/中/高
增长率 ±10%              → [估值区间]
利润率变化 ±5%           → [估值区间]
折现率 ±2%               → [估值区间]
终端增长假设 ±1%         → [估值区间]
```

如无财务预测 → "文档未提供财务预测，无法执行 DCF 分析，仅做市场可比估值"
