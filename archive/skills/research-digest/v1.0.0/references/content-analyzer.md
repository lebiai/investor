# 内容分析逻辑

## 体裁识别

根据内容和结构判断文件属于哪个分类：

| 分类 | 识别特征 |
|------|---------|
| 研报 | 出现"摘要""评级""目标价""买入/卖出""研报""券商"等关键词 |
| 新闻 | 出现"本报讯""记者""报道""快讯""电」"等新闻体特征 |
| 财报 | 包含"资产负债表""利润表""现金流量表""营收""净利润"等财务表格 |
| 纪要 | 出现"会议""发言""Q&A""参会""纪要""讨论决定"等会议特征 |
| 招股书 | 出现"招股说明书""IPO""发行价""募集资金""承销商"等 |
| BP | 出现"商业计划书""BP""business plan""融资计划""项目简介"等 |
| 政策法规 | 出现"第X号""令""通知""管理办法""印发"等公文特征 |
| 白皮书 | 有系统化的技术/行业分析，含大量专业术语、技术细节 |
| 数据报表 | 大量表格/数字结构，以数据为主、文字为辅 |
| 路演材料 | PPT格式，出现"投资亮点""估值""增长策略""谢谢"等路演特征 |
| 合同协议 | 出现"甲方""乙方""本合同""协议""条款""签署"等 |
| 法律文件 | 出现"判决""裁定""诉讼""律师""法律意见"等 |
| 邮件沟通 | 有"发件人""收件人""主题""回复""转发"等邮件格式 |
| 访谈对话 | 一问一答格式，或出现"问：""答：""Q：""A："等 |
| 视频文稿 | 出现时间戳标记，或字幕格式，或明确标注"字幕""转写" |
| 行业资料 | 系统性行业分析但非券商研报格式（学术论文/调研报告等） |
| 内部文档 | 出现"内部""机密""仅供""草案""修订"等内部文档特征 |
| 其它 | 无法匹配以上任何特征 |

## 核心观点提取

- 提取 3-5 条核心观点
- 每条标注置信度（高/中/低）
- 标注来源出处（原文中的段落引用）

## 关键数据提取

- 数值类数据：数字 + 单位 + 时间范围
- 对比类数据：增速变化、市场份额变化
- 估值类数据：PE/PS/市值/估值

## 关联实体识别

识别文本中出现的：
- 公司名 → 查 ../../../data/companies/ 是否有对应档案
- 赛道/行业名 → 查 ../../../data/sectors/
- 技术关键词（如"3nm""Chiplet""固态电池"）
- 人名（创始人/高管/分析师）
