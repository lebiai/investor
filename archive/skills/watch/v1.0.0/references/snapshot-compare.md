# 快照对比逻辑

## 快照格式

每次检查后生成结构化快照，保存到 `data/watch/snapshots/[条目名]_YYYYMMDD.md`：

```markdown
# 快照: [赛道/公司名]
- 类型: 赛道/公司
- 创建时间: YYYY-MM-DD
- 数据来源: [skill:agent-reach] 搜索

## 已知公司（赛道快照时）
| 公司 | 最新估值 | 最近动态时间 | 动态摘要 |
|------|---------|------------|---------|

## 关键指标
- 市场规模：
- 主要参与者数量：
- 最近融资轮次：
- 政策环境变化：

## 来源列表
- [来源1]
- [来源2]
```

## 对比逻辑

新搜索结果 vs 上次快照，逐项比较：

### 新发现（🔴）
- 上次快照中没有的公司/事件
- 新入局者、新融资、新政策

### 变化（🟡）
- 同一公司的估值/营收/指标有变化
- 之前有但状态变了

### 无变化（✅）
- 和上次快照一致的条目
- 输出"无重大变化"即可

### 消失（可能数据问题）
- 上次快照中有但这次没搜到的
- 标注"未找到更新信息"，不自行删除
